# midnight-probe v1.0.2

This release hardens the public diagnostic surface and closes the main failures found during the hostile audit.

Highlights:
- fail-closed rejection of odd-length tx hex input
- metadata submission timeout instead of indefinite hang
- malformed compatibility reports now error instead of being silently treated as missing
- matrix runner now installs with `--ignore-scripts`, filters install environment, injects the full matrix package family, and removes temp workspaces by default
- regression suite expanded to cover the fixed boundaries
- dev-tooling audit cleaned to zero known vulnerabilities after updating vitest to 4.1.4
- npm package `midnight-probe@1.0.2` is now live with `latest` pointing to `1.0.2`
